# Welcome App

Welcome App is a demo UI for login

## Installation

Use the package manager [flutter](https://flutter.dev/docs/get-started/install) to install the project.

```bash
flutter pub get
```

## How to run

```bash
flutter run -d <your_device>
```
Ex: flutter run -d chrome

```bash
flutter run -d chrome
```
